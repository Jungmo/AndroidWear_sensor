# AndroidWear_sensor
Android sensor control - Accelerometer, gyro, heart rate

## Enviroment
* Target Device : LG smart watch Urbane
* Target OS Version : Android 6.0.1
* Android studio SDK Version : API 23 : Android 6.0 (Marshmallow)
